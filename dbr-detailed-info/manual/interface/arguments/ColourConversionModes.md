### BlueChannelWeight
Sets the weight value of Blue Colour Channel used for converting a colour image to a grayscale image.  

- **Value range**   
   [-1, 1000]   
   
- **Default value**   
   -1  
 
- **Remark**     
   -1: The weight value will be set automatically by the SDK.
