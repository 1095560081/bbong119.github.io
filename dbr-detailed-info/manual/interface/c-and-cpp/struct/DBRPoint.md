## DBRPoint
Stores an x- and y-coordinate pair in two-dimensional space.

#### Typedefs

```cpp
typedef struct tagDBRPoint  DBRPoint
```  

#### Attributes
  
| Attribute | Type | Description |
|---------- | ---- | ----------- |
| `x` | *int* | The X coordinate of the point. |
| `y` | *int* | The Y coordinate of the point. |
